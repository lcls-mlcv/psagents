# One-page agent brief

**Goal:** Encode goal, inputs, constraints, and failure behavior for a single ops task.

## Steps (local)

1. Pick a recurring task (e.g. “summarize last night’s log slice”).
2. In one Markdown file, write: goal, allowed inputs, hard constraints, what “done” means.
3. Add one paragraph on what the agent should do if data is missing.

## What you are practicing

Turning an informal request into something you can **evaluate** and **reuse**.

Dummy template; no API keys required.
