# Rubric for “good enough” output

**Goal:** Define 3–5 checkable criteria for success before you run an agent.

## Steps (local)

1. For the same task as the brief example, list criteria (e.g. “mentions timestamp”, “cites source path”).
2. For each criterion, note how you would **verify** it in under a minute.
3. Save next to your brief as `rubric.md`.

## What you are practicing

Evaluation discipline that scales when you move from playground to production.
