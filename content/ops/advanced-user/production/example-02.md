# Compare two prompts (read-only tool)

**Context:** On S3DF you may have access to the same read-only tools from different harnesses or sessions.

## On S3DF (illustrative)

1. Run two phrasings of the same question against a **read-only** tool (e.g. doc search or status query).
2. Paste both outputs into a scratch file and mark differences that matter for ops.
3. Record which phrasing you would put in a SKILL file as the default.

## Take-away

Small prompt changes can change **which** tool path gets chosen; evaluation makes that visible.

> **Dummy:** Use non-destructive commands only; follow site policy.
