# SKILL stub + real CLI on S3DF

**Context:** SKILL files act like man pages for tools that already exist on S3DF.

## On S3DF (illustrative)

1. Pick a CLI that is installed and safe to run read-only (e.g. `sinfo`, `squeue` with no submit).
2. Draft 5 lines of SKILL text: when to use it, one example command, one caveat.
3. Run the example yourself and paste stdout length (not content) into your notes.

## Take-away

If the human cannot run the command, the agent should not pretend it can—fix the **tool path** first.

> **Dummy:** Adjust to your facility’s allowed commands.
